# Rails

Rails are the left and right fixed-width columns that flank the center feed in Davinci's desktop layout. They hold navigational, contextual, and discovery panels that persist as the user scrolls.

## When to use

Rails appear on:
- Home feed page
- Network page
- Jobs page
- Search results

They are hidden on profile pages (which use their own sidebar layout), messaging pages, and mobile breakpoints.

## Left Rail

The left rail is **240px wide** and contains the user's own identity and primary navigation.

### Contents

1. **Profile card** — user's avatar, name, headline, and a link to their profile
2. **NavList** — primary site navigation (Home, Network, Jobs, Notifications, Messages)
3. Optional: **Shortcuts** or **Groups** panel

```html
<aside class="rail rail--left" aria-label="Left sidebar">
  <!-- Identity card -->
  <div class="panel rail__identity">
    <div class="rail__cover" style="background: linear-gradient(135deg, var(--blue-3), var(--sand-3));"></div>
    <div class="rail__profile">
      <div class="avatar avatar--lg">
        <img src="/me.jpg" alt="My profile" />
      </div>
      <div class="rail__profile-info">
        <a class="rail__profile-name" href="/in/me">Jordan Reeves</a>
        <p class="rail__profile-headline">Product Designer</p>
      </div>
    </div>
    <div class="rail__stats">
      <div class="rail__stat">
        <span class="rail__stat-label">Profile views</span>
        <span class="rail__stat-value" style="color: var(--accent-fg);">142</span>
      </div>
```

### Token usage

| Property | Token |
|---|---|
| Width | 240px |
| Gap between panels | `--s-5` |
| Cover height | 64px |
| Profile name | `--text-sm` / `--fw-semibold` |
| Headline | `--text-xs` / `--fg-muted` |
| Stat label | `--text-xs` / `--fg-subtle` |
| Stat value | `--text-sm` / `--fw-semibold` / `--accent-fg` |

## Right Rail

The right rail is **300–320px wide** and contains discovery and contextual content — ads, suggested connections, trending topics.

### Contents

1. **Ad unit** — sponsored content panel (labeled "Promoted")
2. **People you may know** — connection suggestions panel
3. **Trending** — optional trending topics or company news


## Sticky behavior

Both rails use `position: sticky; top: calc(56px + var(--s-5))` to stay fixed relative to the viewport as the center column scrolls. The `56px` accounts for the TopNav height.

If rail contents are taller than the viewport, the rail scrolls independently with `overflow-y: auto` and hidden scrollbars.

## Responsive

| Breakpoint | Behavior |
|---|---|
| ≥ 1200px | Both rails visible |
| 900–1199px | Left rail only; right rail hidden |
| < 900px | Both rails hidden; navigation moves to bottom tab bar |
