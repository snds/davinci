# Textarea

Textarea is the multiline text field primitive. It inherits the same border, focus ring, and token system as `.input`.

## HTML structure

```html
<div class="form-group">
  <label class="label">About you</label>
  <textarea class="textarea" rows="4" placeholder="Describe yourself…"></textarea>
  <span class="form-hint">Max 300 characters</span>
</div>
```
