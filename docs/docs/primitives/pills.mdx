# Pills

Pills are compact, inline status indicators, category tags, and count badges. They are always `--radius-full` (fully rounded) and use the `--text-xs` type scale.

## Variants

| Class | Background | Text | Border | Use case |
|---|---|---|---|---|
| `.pill` | `--bg-subtle` | `--fg-muted` | `--border-subtle` | Neutral category, default tag |
| `.pill--accent` | `--accent-subtle` | `--accent-fg` | `--accent-border` | Selected, active, featured |
| `.pill--alt` | `--alt-subtle` | `--alt-fg` | — | Brand highlight, new, featured |
| `.pill--success` | `--green-a3` | `--success-fg` | rgba green | Active status, positive indicator |

## Base anatomy

A pill consists of an optional leading icon, label text, and an optional trailing count. All parts are independently optional.

```html
<!-- Label only -->
<span class="pill">Design</span>

<!-- Icon + label -->
<span class="pill pill--accent">
  <span class="material-symbols-rounded" style="font-size:12px;">star</span>
  Featured
</span>

<!-- Label + count -->
<span class="pill">
  Connections
  <span style="opacity:0.6;margin-left:2px;">· 248</span>
</span>
```

<DocsCallout type="info">
Pills are display-only. For interactive filter-style pills, derive from `.btn--pill` and add focus, selected, and active states. Never make a plain `.pill` the primary interactive trigger.
</DocsCallout>
