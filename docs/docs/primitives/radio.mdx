## HTML structure

```html
<fieldset>
  <legend>Job type</legend>
  <label class="radio-wrap">
    <input type="radio" name="job-type" checked />
    <span>Full-time</span>
  </label>
  <label class="radio-wrap">
    <input type="radio" name="job-type" />
    <span>Part-time</span>
  </label>
</fieldset>
```

<DocsCallout type="tip" title="Accessibility">
  Always use the same `name` attribute on related radio inputs so the browser handles mutual exclusion. Wrap groups in a `<fieldset>` + `<legend>`.
</DocsCallout>
