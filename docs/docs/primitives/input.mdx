# Input

Input is the single-line text field primitive. It pairs with `.label` and `.form-hint` to build accessible form groups.

  title="Input variants"

## HTML structure

```html
<!-- Basic form group -->
<div class="form-group">
  <label class="label label--required">Full name</label>
  <input class="input" type="text" placeholder="Jane Doe" />
  <span class="form-hint">Appears on your public profile</span>
</div>

<!-- Input with icon -->
<div class="input-wrap">
  <span class="material-symbols-rounded input-icon">search</span>
  <input class="input" type="text" placeholder="Search…" />
</div>

<!-- Error state -->
<div class="form-group">
  <label class="label">Email</label>
  <input class="input input--error" type="email" value="not-an-email" />
  <span class="form-error">
    <span class="material-symbols-rounded" style="font-size:13px;">error</span>
    Please enter a valid email address
  </span>
</div>
```
