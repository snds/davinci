# Slider

Slider is a range input styled with an accent-colored thumb and a neutral track. It is purely CSS — no JS required for basic use.

## HTML structure

```html
<div class="form-group">
  <label class="label">Salary range</label>
  <input class="slider" type="range" min="40" max="200" value="80" />
</div>
```

<DocsCallout type="info">
  Read the value in JS via `input.value`. Pair with a live output element (`<output>`) to display the current value accessibly.
</DocsCallout>
