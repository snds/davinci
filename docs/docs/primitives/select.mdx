# Select

Select wraps a native `<select>` with Davinci surface styling, a custom chevron, and a consistent focus ring — no JS required.

## HTML structure

```html

  <label class="label">Industry</label>
  <div class="select-wrap">
    <select class="select">
      <option value="">Choose…</option>
      <option>Technology</option>
    </select>
  </div>
</div>
```

## CSS classes

<PropsTable props={[
  { name: '.select-wrap', type: 'div', description: 'Relative wrapper that injects the custom chevron via ::after' },

<DocsCallout type="info">
  `.select-wrap::after` uses the Material Symbols `expand_more` character. If you're not loading Material Symbols, replace it with a CSS-drawn chevron SVG background image.
</DocsCallout>
