# Checkbox

## HTML structure

```html
<label class="checkbox-wrap">
  <input type="checkbox" checked />
  <span>Remote positions only</span>
</label>
```

<DocsCallout type="tip" title="Accessibility">
  Wrapping `<input>` and `<span>` inside a `<label>` makes the entire row clickable without any JS. For groups of checkboxes, use a `<fieldset>` with a `<legend>` to provide a group label to screen readers.
</DocsCallout>
