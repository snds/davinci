## HTML structure

```html
<!-- Text line -->
<div class="skeleton skeleton--text" style="width: 80%;"></div>

<!-- Heading -->
<div class="skeleton skeleton--heading" style="width: 50%;"></div>

<!-- Circle (avatar) -->
<div class="skeleton skeleton--circle" style="width: 40px; height: 40px;"></div>

<!-- Block (image/card cover) -->
<div class="skeleton" style="height: 120px; border-radius: 8px;"></div>
```

<DocsCallout type="tip">
  Match skeleton dimensions as closely as possible to the real content. This reduces layout shift when content loads and makes the transition feel seamless.
</DocsCallout>
