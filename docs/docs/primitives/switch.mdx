# Switch

Switch is a binary on/off toggle with a sliding thumb. Use it for settings that take effect immediately — unlike a checkbox, a switch implies instant action, not form submission.

## HTML structure

```html
<!-- Off state -->
<label class="switch">
  <div class="switch__track">
    <div class="switch__thumb"></div>
  </div>
  <span>Email notifications</span>
</label>

<!-- On state — add switch--on to the root label -->
<label class="switch switch--on">
  <div class="switch__track">
    <div class="switch__thumb"></div>
  </div>
  <span>Push notifications</span>
</label>
```

<DocsCallout type="info">
  The on/off state is CSS-driven via `.switch--on`. Toggle this class with JS on click. For accessibility, add `role="switch"` and `aria-checked="true/false"` to the root element.
</DocsCallout>
