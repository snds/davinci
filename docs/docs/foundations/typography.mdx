# Typography

Davinci's type system is built on **Inter** for UI text and **JetBrains Mono** for code. A 13-step type scale covers every context from dense micro-labels to large display headings, with matching line-height and letter-spacing tokens.

## Font families

Both display and body use the same font stack: Inter → system-ui → -apple-system → Segoe UI → Roboto. The distinction exists for semantic clarity and potential future differentiation.

## Heading defaults

The base CSS in `colors_and_type.css` applies sensible defaults to HTML heading elements:

| Element | Size | Weight | Line height | Letter spacing |
|---|---|---|---|---|
| `h1` | `--text-5xl` | `--fw-heavy` | `--lh-tight` | `--ls-tight` |
| `h2` | `--text-3xl` | `--fw-bold` | `--lh-snug` | `--ls-snug` |
| `h3` | `--text-xl` | `--fw-semibold` | `--lh-snug` | `--ls-snug` |
| `h4` | `--text-lg` | `--fw-semibold` | `--lh-normal` | `--ls-normal` |
| `h5` | `--text-md` | `--fw-semibold` | `--lh-normal` | `--ls-normal` |
| `h6` | `--text-sm` | `--fw-semibold` | `--lh-normal` | `--ls-wide` |

## Usage guidelines

**Body text** — use `--text-base` / `--fw-regular` / `--lh-normal` for most content. Switch to `--text-md` in dense layouts (sidebars, tables, forms).

**Labels and captions** — `--text-xs` or `--text-sm` with `--fw-medium` and `--ls-wide` for uppercase labels. Use `--fg-muted` for secondary labels, `--fg-subtle` for tertiary.

**Code** — always use `--font-mono`. Default to `--text-sm`. Pair with `--bg-surface` background and `--border-subtle` border for inline code.

**Headings** — follow the table above. Avoid skipping levels (e.g. h1 → h3). Letter spacing at large sizes (-0.02em) prevents Inter from feeling too loose.

**Don't** override fonts with system fonts in components — Inter provides a consistent platform feel.
