# Skeleton

Pulse-animated placeholder used to communicate loading state and prevent layout shift. No Radix dependency — just a `div` with an animated background pulse. Size it to match the real content it replaces.

```bash
cd packages/ui && npx shadcn@latest add skeleton
```

## Usage

```jsx
import { Skeleton } from '@davinci/ui/components/ui/skeleton';

// Card with header and body lines
<div className="flex flex-col gap-3 p-4">
  <Skeleton className="h-40 w-full rounded-md" />
  <Skeleton className="h-4 w-[250px]" />
  <Skeleton className="h-4 w-[200px]" />
  <Skeleton className="h-4 w-[175px]" />
</div>

// Avatar + text row
<div className="flex items-center gap-3">
  <Skeleton className="h-10 w-10 rounded-full" />
  <div className="flex flex-col gap-2">
    <Skeleton className="h-4 w-[150px]" />
    <Skeleton className="h-3 w-[100px]" />
  </div>
</div>
```

Size with `h-*` and `w-*` utilities to match the real content dimensions exactly.
