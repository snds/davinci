# Resizable

Drag-to-resize panel groups for split-view layouts. Built on `react-resizable-panels`. Use it for editor+preview splits, sidebar+content layouts, and any UI where users should be able to control pane proportions.

```bash
cd packages/ui && npx shadcn@latest add resizable
```

## Sub-components

| Component | Role |
|-----------|------|
| `ResizablePanelGroup` | Root — sets layout `direction` |
| `ResizablePanel` | Individual resizable pane |
| `ResizableHandle` | Drag handle between panels |

## Usage

```jsx
import {
  ResizablePanelGroup,
  ResizablePanel,
  ResizableHandle,
} from '@davinci/ui/components/ui/resizable';

// Horizontal split
<ResizablePanelGroup direction="horizontal" className="min-h-[200px] rounded-lg border">
  <ResizablePanel defaultSize={25}>
    <div className="flex h-full items-center justify-center p-4">Sidebar</div>
  </ResizablePanel>
  <ResizableHandle withHandle />
  <ResizablePanel defaultSize={75}>
    <div className="flex h-full items-center justify-center p-4">Content</div>
  </ResizablePanel>
</ResizablePanelGroup>

// Vertical split
<ResizablePanelGroup direction="vertical" className="min-h-[400px]">
  <ResizablePanel defaultSize={50}>Top</ResizablePanel>
  <ResizableHandle withHandle />
  <ResizablePanel defaultSize={50}>Bottom</ResizablePanel>
</ResizablePanelGroup>
```

Pass `withHandle` to `ResizableHandle` to show a visible grip icon. `defaultSize` is a percentage (0–100); sibling panels should sum to 100.
