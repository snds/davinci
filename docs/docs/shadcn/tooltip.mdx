# Tooltip

Built on `@radix-ui/react-tooltip`. Appears on hover or keyboard focus with a configurable delay. Renders in a portal; never clipped. Must be wrapped in `TooltipProvider` — typically done once at the app root.

```bash
cd packages/ui && npx shadcn@latest add tooltip
```

## Sub-components

| Component | Role |
|-----------|------|
| `TooltipProvider` | Sets delay and skip-delay for all tooltips — wrap the app root |
| `Tooltip` | Per-instance context |
| `TooltipTrigger` | The element that shows the tooltip on hover/focus |
| `TooltipContent` | The floating label |

## Usage

```jsx
import {
  TooltipProvider, Tooltip, TooltipTrigger, TooltipContent,
} from '@davinci/ui/components/ui/tooltip';
import { Button } from '@davinci/ui/components/ui/button';

// Wrap once at app/layout level
<TooltipProvider>
  <Tooltip>
    <TooltipTrigger asChild>
      <Button variant="ghost" size="icon"><InfoIcon /></Button>
    </TooltipTrigger>
    <TooltipContent>
      <p>Add to library</p>
    </TooltipContent>
  </Tooltip>
</TooltipProvider>
```

## Side and alignment

```jsx
<TooltipContent side="right" align="start">
  Shows to the right, aligned to the start
</TooltipContent>
```

`side`: `top` | `right` | `bottom` | `left`  
`align`: `start` | `center` | `end`
