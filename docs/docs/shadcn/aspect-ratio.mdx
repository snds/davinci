# Aspect Ratio

A layout utility that forces its child to maintain a specific width-to-height ratio regardless of available space. Built on `@radix-ui/react-aspect-ratio`. Use it for image thumbnails, video embeds, map tiles, and any content that must stay proportional as its container resizes.

```bash
cd packages/ui && npx shadcn@latest add aspect-ratio
```

## Usage

```jsx
import { AspectRatio } from '@davinci/ui/components/ui/aspect-ratio';

{/* 16:9 video thumbnail */}
<div className="w-full max-w-sm">
  <AspectRatio ratio={16 / 9}>
    <img
      src="/thumbnail.jpg"
      alt="Video thumbnail"
      className="h-full w-full rounded-md object-cover"
    />
  </AspectRatio>
</div>

{/* Square avatar */}
<div className="w-16">
  <AspectRatio ratio={1}>
    <img src="/avatar.jpg" alt="Avatar" className="h-full w-full rounded-full object-cover" />
  </AspectRatio>
</div>
```

The `ratio` prop accepts any number (default `1`). Use fractional expressions like `16/9`, `4/3`, or `1/1` for clarity. The child must fill the full width and height of the container (`h-full w-full`).
