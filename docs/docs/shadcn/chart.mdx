# Chart

A thin wrapper around `recharts` that injects Davinci design tokens into charts via a `ChartConfig` object. Provides a consistent container, accessible tooltip, and legend components that all read from the same config.

```bash
cd packages/ui && npx shadcn@latest add chart
```

## Sub-components

| Component | Role |
|-----------|------|
| `ChartContainer` | Root wrapper — accepts `config` and injects CSS vars |
| `ChartTooltip` | Recharts `<Tooltip>` pre-wired to Davinci styles |
| `ChartTooltipContent` | Tooltip body — renders label, value, and color swatch |
| `ChartLegend` | Recharts `<Legend>` pre-wired to Davinci styles |
| `ChartLegendContent` | Legend body — renders color swatch and series name |

## Usage

```jsx
import { Bar, BarChart, CartesianGrid, XAxis } from 'recharts';
import {
  ChartContainer, ChartTooltip, ChartTooltipContent,
  ChartLegend, ChartLegendContent,
} from '@davinci/ui/components/ui/chart';
import type { ChartConfig } from '@davinci/ui/components/ui/chart';

const chartData = [
  { month: 'Jan', desktop: 186, mobile: 80 },
  { month: 'Feb', desktop: 305, mobile: 200 },
  { month: 'Mar', desktop: 237, mobile: 120 },
];

const chartConfig = {
  desktop: { label: 'Desktop', color: 'var(--chart-1)' },
  mobile:  { label: 'Mobile',  color: 'var(--chart-2)' },
} satisfies ChartConfig;

export function BarChartDemo() {
  return (
    <ChartContainer config={chartConfig} className="min-h-[200px] w-full">
      <BarChart data={chartData}>
        <CartesianGrid vertical={false} />
        <XAxis dataKey="month" tickLine={false} axisLine={false} />
        <ChartTooltip content={<ChartTooltipContent />} />
        <ChartLegend content={<ChartLegendContent />} />
        <Bar dataKey="desktop" fill="var(--color-desktop)" radius={4} />
        <Bar dataKey="mobile"  fill="var(--color-mobile)"  radius={4} />
      </BarChart>
    </ChartContainer>
  );
}
```

## Theming

`ChartContainer` injects `--color-<key>` CSS variables for each entry in `config`, sourced from the `color` field. Use `var(--chart-1)` through `var(--chart-5)` as the Davinci palette defaults, or supply any arbitrary color.

| Token | Purpose |
|-------|---------|
| `--chart-1` … `--chart-5` | Default series color ramp |
| `--bg-elevated` | Tooltip panel background |
| `--border` | Grid lines and tooltip border |
