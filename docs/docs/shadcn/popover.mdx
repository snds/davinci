# Popover

Built on `@radix-ui/react-popover`. A non-modal floating panel anchored to a trigger. Use for pickers, inline forms, and detail previews that don't require blocking the rest of the UI.

```bash
cd packages/ui && npx shadcn@latest add popover
```

## Sub-components

| Component | Role |
|-----------|------|
| `Popover` | Root context |
| `PopoverTrigger` | The element that opens the popover |
| `PopoverContent` | The floating panel |
| `PopoverAnchor` | Optional explicit anchor (defaults to trigger) |

## Usage

```jsx
import { Popover, PopoverTrigger, PopoverContent } from '@davinci/ui/components/ui/popover';
import { Button } from '@davinci/ui/components/ui/button';
import { Input } from '@davinci/ui/components/davinci/input';
import { Label } from '@davinci/ui/components/ui/label';

<Popover>
  <PopoverTrigger asChild>
    <Button variant="outline">Edit dimensions</Button>
  </PopoverTrigger>
  <PopoverContent className="w-80">
    <div className="grid gap-4">
      <div className="space-y-2">
        <h4 className="font-medium">Dimensions</h4>
        <p className="text-sm text-muted-foreground">Set the container dimensions.</p>
      </div>
      <div className="grid gap-2">
        <div className="grid grid-cols-3 items-center gap-4">
          <Label>Width</Label>
          <Input className="col-span-2" defaultValue="100%" />
        </div>
        <div className="grid grid-cols-3 items-center gap-4">
          <Label>Height</Label>
          <Input className="col-span-2" defaultValue="auto" />
        </div>
      </div>
    </div>
  </PopoverContent>
</Popover>
```

## Alignment and side

```jsx
<PopoverContent side="bottom" align="start" sideOffset={8}>
  …
</PopoverContent>
```
