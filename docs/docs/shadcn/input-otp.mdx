# Input OTP

A segmented one-time password (OTP) input with full keyboard navigation and paste support. Built on the `input-otp` library. Each character occupies its own visual slot, making it suitable for verification codes, PINs, and access codes.

```bash
cd packages/ui && npx shadcn@latest add input-otp
```

## Sub-components

| Component | Role |
|-----------|------|
| `InputOTP` | Root — manages value, maxLength, and pattern |
| `InputOTPGroup` | Groups adjacent slots visually |
| `InputOTPSlot` | A single character cell (receives `index` prop) |
| `InputOTPSeparator` | Visual divider between slot groups |

## Usage

```jsx
import {
  InputOTP, InputOTPGroup, InputOTPSlot, InputOTPSeparator,
} from '@davinci/ui/components/ui/input-otp';

{/* 6-digit numeric code */}
<InputOTP maxLength={6}>
  <InputOTPGroup>
    <InputOTPSlot index={0} />
    <InputOTPSlot index={1} />
    <InputOTPSlot index={2} />
  </InputOTPGroup>
  <InputOTPSeparator />
  <InputOTPGroup>
    <InputOTPSlot index={3} />
    <InputOTPSlot index={4} />
    <InputOTPSlot index={5} />
  </InputOTPGroup>
</InputOTP>
```

### Controlled with pattern

```jsx
import { REGEXP_ONLY_DIGITS_AND_CHARS } from 'input-otp';

const [value, setValue] = React.useState('');

<InputOTP
  maxLength={6}
  pattern={REGEXP_ONLY_DIGITS_AND_CHARS}
  value={value}
  onChange={setValue}
>
  <InputOTPGroup>
    {Array.from({ length: 6 }).map((_, i) => (
      <InputOTPSlot key={i} index={i} />
    ))}
  </InputOTPGroup>
</InputOTP>
```

## Key props on `InputOTP`

| Prop | Type | Description |
|------|------|-------------|
| `maxLength` | `number` | Total number of characters |
| `pattern` | `string` (regex) | Allowed character class — use exported constants like `REGEXP_ONLY_DIGITS`, `REGEXP_ONLY_CHARS`, `REGEXP_ONLY_DIGITS_AND_CHARS` |
| `value` | `string` | Controlled value |
| `onChange` | function | Called on every character change |
| `disabled` | `boolean` | Disables the entire input |
