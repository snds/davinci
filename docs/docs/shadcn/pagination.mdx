# Pagination

Composable page navigation built from semantic `<nav>` and `<a>` elements. No Radix dependency.

```bash
cd packages/ui && npx shadcn@latest add pagination
```

## Sub-components

| Component | Role |
|-----------|------|
| `Pagination` | Root `<nav>` wrapper |
| `PaginationContent` | Flex container for items |
| `PaginationItem` | Wrapper for each control |
| `PaginationLink` | Individual page link ŌĆö accepts `isActive` |
| `PaginationPrevious` | Previous-page chevron link |
| `PaginationNext` | Next-page chevron link |
| `PaginationEllipsis` | `ŌĆ”` placeholder for skipped pages |

## Usage

```jsx
import {
  Pagination,
  PaginationContent,
  PaginationItem,
  PaginationLink,
  PaginationPrevious,
  PaginationNext,
  PaginationEllipsis,
} from '@davinci/ui/components/ui/pagination';

<Pagination>
  <PaginationContent>
    <PaginationItem>
      <PaginationPrevious href="?page=2" />
    </PaginationItem>

    <PaginationItem>
      <PaginationLink href="?page=1">1</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="?page=2">2</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="?page=3" isActive>3</PaginationLink>
    </PaginationItem>

    <PaginationItem>
      <PaginationEllipsis />
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="?page=10">10</PaginationLink>
    </PaginationItem>

    <PaginationItem>
      <PaginationNext href="?page=4" />
    </PaginationItem>
  </PaginationContent>
</Pagination>
```
