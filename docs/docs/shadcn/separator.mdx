# Separator

A thin line that visually or semantically separates content. Built on `@radix-ui/react-separator`. Supports horizontal and vertical orientations and can be marked decorative to hide it from assistive technologies.

```bash
cd packages/ui && npx shadcn@latest add separator
```

## Usage

```jsx
import { Separator } from '@davinci/ui/components/ui/separator';

// Horizontal (default)
<Separator />

// Vertical — needs explicit height on parent
<div className="flex h-5 items-center gap-4">
  <span>Posts</span>
  <Separator orientation="vertical" />
  <span>Comments</span>
  <Separator orientation="vertical" />
  <span>Likes</span>
</div>

// Decorative (no ARIA role)
<Separator decorative />
```

## Theming

Uses `--border-subtle` (sand-4). Override with `className="bg-border"` or any Tailwind color utility.
