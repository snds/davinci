# Dialog

A modal window built on `@radix-ui/react-dialog`. Provides focus trapping, scroll locking, and full keyboard/screen-reader support. Use for actions that require user decision before proceeding.

```bash
cd packages/ui && npx shadcn@latest add dialog
```

## Sub-components

| Component | Role |
|-----------|------|
| `Dialog` | Root context provider |
| `DialogTrigger` | Button that opens the dialog |
| `DialogContent` | The modal panel — renders in a portal |
| `DialogHeader` | Top section inside DialogContent |
| `DialogTitle` | Required — names the dialog for accessibility |
| `DialogDescription` | Optional subtitle / helper text |
| `DialogFooter` | Bottom section — action buttons |
| `DialogClose` | Button that dismisses the dialog |

## Usage

```jsx
import {
  Dialog, DialogTrigger, DialogContent, DialogHeader,
  DialogTitle, DialogDescription, DialogFooter, DialogClose,
} from '@davinci/ui/components/ui/dialog';
import { Button } from '@davinci/ui/components/ui/button';

<Dialog>
  <DialogTrigger asChild>
    <Button variant="outline">Edit profile</Button>
  </DialogTrigger>
  <DialogContent className="sm:max-w-[425px]">
    <DialogHeader>
      <DialogTitle>Edit profile</DialogTitle>
      <DialogDescription>
        Make changes to your profile here. Click save when done.
      </DialogDescription>
    </DialogHeader>
    {/* form fields */}
    <DialogFooter>
      <DialogClose asChild>
        <Button variant="ghost">Cancel</Button>
      </DialogClose>
      <Button type="submit">Save changes</Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```

## Controlled (no trigger button)

```jsx
const [open, setOpen] = React.useState(false);

<Dialog open={open} onOpenChange={setOpen}>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>Confirm deletion</DialogTitle>
    </DialogHeader>
    <DialogFooter>
      <Button variant="ghost" onClick={() => setOpen(false)}>Cancel</Button>

    </DialogFooter>

```
