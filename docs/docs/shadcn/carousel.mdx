# Carousel

A touch-friendly, accessible carousel component built on `embla-carousel-react`. Supports horizontal and vertical orientation, custom scroll options, and autoplay via Embla plugins.

```bash
cd packages/ui && npx shadcn@latest add carousel
```

## Sub-components

| Component | Role |
|-----------|------|
| `Carousel` | Root provider — accepts `orientation` and `opts` |
| `CarouselContent` | The scrollable track that holds items |
| `CarouselItem` | An individual slide — use `basis-*` to control width |
| `CarouselPrevious` | Previous button (arrow left) |
| `CarouselNext` | Next button (arrow right) |

## Usage

```jsx
import {
  Carousel, CarouselContent, CarouselItem, CarouselPrevious, CarouselNext,
} from '@davinci/ui/components/ui/carousel';
import { Card, CardContent } from '@davinci/ui/components/ui/card';

<Carousel className="w-full max-w-sm">
  <CarouselContent>
    {Array.from({ length: 5 }).map((_, index) => (
      <CarouselItem key={index}>
        <Card>
          <CardContent className="flex aspect-square items-center justify-center p-6">
            <span className="text-4xl font-semibold">{index + 1}</span>
          </CardContent>
        </Card>
      </CarouselItem>
    ))}
  </CarouselContent>
  <CarouselPrevious />
  <CarouselNext />
</Carousel>
```

### Multiple visible items

```jsx
<Carousel opts={{ align: 'start' }}>
  <CarouselContent className="-ml-4">
    {items.map((item) => (
      <CarouselItem key={item.id} className="pl-4 md:basis-1/2 lg:basis-1/3">
        {/* card content */}
      </CarouselItem>
    ))}
  </CarouselContent>
</Carousel>
```

### Vertical carousel

```jsx
<Carousel orientation="vertical" className="h-64">
  <CarouselContent className="-mt-4 h-64">
    {items.map((item) => (
      <CarouselItem key={item.id} className="pt-4 basis-1/3">
        {/* content */}
      </CarouselItem>
    ))}
  </CarouselContent>
  <CarouselPrevious />
  <CarouselNext />
</Carousel>
```

## Key props

| Prop | Type | Description |
|------|------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | Scroll direction |
| `opts` | `EmblaOptionsType` | Passed directly to Embla (loop, align, skipSnaps, etc.) |
| `plugins` | `EmblaPluginType[]` | Embla plugins (e.g. Autoplay) |
