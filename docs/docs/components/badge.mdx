# Badge

Badge is a compact numeric indicator used to surface unread counts on nav icons or notification bells. It is distinct from `.pill` — pills label categories; badges surface counts.

## HTML structure

```html
<!-- On an icon -->
<div style="position: relative; display: inline-flex;">
  <span class="material-symbols-rounded">notifications</span>
  <span class="badge" style="position: absolute; top: -6px; right: -8px;">3</span>
</div>

<!-- Dot badge -->
<div style="position: relative; display: inline-flex;">
  <span class="material-symbols-rounded">person</span>
  <span class="badge badge--dot" style="position: absolute; top: -2px; right: -2px;"></span>
</div>

<!-- Inline neutral -->
<span class="badge badge--neutral">42</span>
```
