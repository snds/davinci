# Alert

Alert is a semantic inline banner for status messages — info, success, warning, and danger. Unlike a Toast, an Alert is persistent and inline within the page content.

## HTML structure

```html
<div class="alert alert--info">
  <span class="material-symbols-rounded alert__icon">info</span>
  <div class="alert__body">
    <div class="alert__title">Heads up</div>
    <div class="alert__desc">Additional context goes here.</div>
  </div>
</div>
```
