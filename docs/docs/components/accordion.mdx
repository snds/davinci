# Accordion

Accordion collapses and expands sections of content. Add `.accordion__item--open` to a section to reveal its content. Toggle with JS in production.

<LiveDemo
  title="Accordion"
  html={`<div class="accordion" style="max-width:480px;">
  <div class="accordion__item accordion__item--open">
    <button class="accordion__trigger" type="button">
      What is Davinci Design System?
      <span class="material-symbols-rounded accordion__icon">expand_more</span>
    </button>
    <div class="accordion__content">
      Davinci is a CSS-first design system built for professional social networks. It uses Radix-based color tokens, display + body font pairing, and a component layer that maps directly to real product UI.
    </div>
  </div>
  <div class="accordion__item">
    <button class="accordion__trigger" type="button">
      How do I use it without a framework?
      <span class="material-symbols-rounded accordion__icon">expand_more</span>
    </button>
    <div class="accordion__content" style="display:none;">
      Just link the CSS file and use the class names in plain HTML. No JavaScript framework is required — interactivity (dropdowns, tabs) can be added with vanilla JS event listeners.
    </div>
  </div>
  <div class="accordion__item">
    <button class="accordion__trigger" type="button">
      Does it support dark mode?

    </div>
  </div>
</div>`}

## HTML structure

```html
<div class="accordion">
  <!-- Open item -->
  <div class="accordion__item accordion__item--open">
    <button class="accordion__trigger" type="button">
      Question text
      <span class="material-symbols-rounded accordion__icon">expand_more</span>
    </button>
    <div class="accordion__content">
      Answer content here.
    </div>
  </div>

  <!-- Closed item -->
  <div class="accordion__item">
    <button class="accordion__trigger" type="button">
      Another question
      <span class="material-symbols-rounded accordion__icon">expand_more</span>
    </button>
    <div class="accordion__content" style="display: none;">
      Hidden answer content.
    </div>
  </div>
</div>
```
