# Dialog

Dialog (modal) overlays a focused task above the page. It uses `.dialog-overlay` as a backdrop and `.dialog` as the centered panel. Nested components: Button, Input, Alert.

## HTML structure

```html
<!-- Overlay -->
<div class="dialog-overlay">
  <div class="dialog">
    <div class="dialog__header">
      <div class="dialog__title">Dialog title</div>
      <p class="dialog__desc">Supporting description.</p>
    </div>
    <div class="dialog__body">
      <!-- form fields, alert, or content -->
    </div>
    <div class="dialog__footer">
      <button class="btn btn--secondary">Cancel</button>
      <button class="btn btn--primary">Confirm</button>
    </div>
  </div>
</div>
```

<DocsCallout type="tip" title="Accessibility">
  Add `role="dialog"`, `aria-modal="true"`, and `aria-labelledby` pointing at the title. Trap focus inside the dialog while open. Return focus to the trigger element on close.
</DocsCallout>
