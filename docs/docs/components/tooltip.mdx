# Tooltip

Tooltip surfaces a short label on hover or focus. It is purely CSS — no JS required for the positioning, though you'll need JS to manage `aria-describedby` and show/hide on keyboard focus.

## HTML structure

```html
<!-- Always visible (CSS demo) — in production, show/hide with JS -->
<div class="tooltip-wrap">
  <div class="tooltip">Tooltip text</div>
  <button class="btn btn--ghost btn--icon">
    <span class="material-symbols-rounded">more_horiz</span>
  </button>
</div>

<!-- Bottom placement -->
<div class="tooltip-wrap">
  <button class="btn btn--secondary">Hover</button>
  <div class="tooltip tooltip--bottom">Below the button</div>
</div>
```

<DocsCallout type="note">
  The `.tooltip` is always rendered in the DOM but hidden via `pointer-events: none`. In production, use JS to toggle visibility on `mouseenter`/`mouseleave` and `focus`/`blur`. Add `aria-describedby` pointing to the tooltip's `id`.
</DocsCallout>
