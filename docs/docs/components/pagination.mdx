# Pagination

Pagination navigates between pages of content. The active page gets `.active` styling; disabled prev/next get the `:disabled` state.

## HTML structure

```html
<nav class="pagination" aria-label="Page navigation">
  <button class="page-btn" disabled>
    <span class="material-symbols-rounded">chevron_left</span>
  </button>
  <button class="page-btn active" aria-current="page">1</button>
  <button class="page-btn">2</button>
  <button class="page-btn">3</button>
  <button class="page-btn">
    <span class="material-symbols-rounded">chevron_right</span>
  </button>
</nav>
```

<DocsCallout type="tip" title="Accessibility">
  Wrap in `<nav aria-label="Page navigation">`. Add `aria-current="page"` to the active page button. Label the prev/next icon buttons with `aria-label="Previous page"` and `aria-label="Next page"`.
</DocsCallout>
