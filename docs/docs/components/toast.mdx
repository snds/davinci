# Toast

Toast is an ephemeral notification that appears at the edge of the screen and auto-dismisses. Unlike Alert (inline and persistent), a toast interrupts briefly and disappears.

## HTML structure

```html
<div class="toast toast--success">
  <span class="material-symbols-rounded toast__icon">check_circle</span>
  <div class="toast__body">
    <div class="toast__title">Saved</div>
    <div class="toast__desc">Your changes were saved.</div>
  </div>
  <span class="material-symbols-rounded toast__close">close</span>
</div>
```

**Positioning (fixed, bottom-right):**
```css
.toast-region {
  position: fixed;
  bottom: 24px;
  right: 24px;
  z-index: 400;
  display: flex;
  flex-direction: column;
  gap: 8px;
}
```
