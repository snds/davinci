# Toggle

Toggle is a pressable icon button with an active state — used for formatting controls, filter chips, and view-switcher groups. Distinct from Switch (which signals on/off system state).

## HTML structure

```html
<!-- Single toggle -->
<button class="toggle active">
  <span class="material-symbols-rounded">format_bold</span>
</button>

<!-- Toggle group -->
<div class="toggle-group">
  <button class="toggle active">
    <span class="material-symbols-rounded">view_list</span>
  </button>
  <button class="toggle">
    <span class="material-symbols-rounded">grid_view</span>
  </button>
</div>
```
