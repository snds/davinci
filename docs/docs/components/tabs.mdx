# Tabs

Tabs switch between related views within the same context. Two variants are available: underline tabs (default) and pill tabs for contained contexts.

## HTML structure

```html
<!-- Underline tabs -->
<div class="tab-list" role="tablist">
  <button class="tab active" role="tab">Posts</button>
  <button class="tab" role="tab">Activity</button>
</div>
<div class="tab-panel" role="tabpanel">
  <!-- panel content -->
</div>

<!-- Pill tabs -->
<div class="tab-list tab-list--pills" role="tablist">
  <button class="tab active" role="tab">Overview</button>
  <button class="tab" role="tab">Experience</button>
</div>
```

<DocsCallout type="tip" title="Accessibility">
  Add `role="tablist"` to the container, `role="tab"` + `aria-selected` to each tab, and `role="tabpanel"` + `aria-labelledby` to each panel. Manage keyboard navigation (Arrow keys, Home, End) with JS.
</DocsCallout>
