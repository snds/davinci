# Panel

Panel is the primary surface container in Davinci. It wraps any bounded region of related content — a profile summary, a list of connections, an ad unit, a settings section. It provides consistent spacing, surface color, border, and optional header/footer regions.

## Anatomy

```
┌─────────────────────────────────┐  ← border: 1px solid --border
│ Panel Header                    │  ← .panel__header
│  Title          [Action button] │
├─────────────────────────────────┤  ← border-bottom: 1px solid --border
│                                 │
│  Panel Body                     │  ← .panel__body
│  (slot for any content)         │
│                                 │
└─────────────────────────────────┘
```

## HTML structure

```html
<!-- Panel with header -->
<div class="panel">
  <div class="panel__header">
    <span>Section Title</span>
    <button class="btn btn--ghost btn--sm">Action</button>
  </div>
  <div class="panel__body">
    <!-- any content -->
  </div>
</div>

<!-- Minimal panel — no header -->
<div class="panel">
  <div class="panel__body">
    <!-- any content -->
  </div>
</div>
```

## Stacking panels

Panels stacked vertically in a sidebar automatically get `16px` top margin via the `panel + panel` adjacency selector.

<DocsCallout type="info">
The `overflow: hidden` on `.panel` ensures child elements respect the border-radius. If you need overflow-visible for dropdowns or tooltips, render those outside the panel or use a portal.
</DocsCallout>
