# Breadcrumb

Breadcrumb shows the user's location in a hierarchy. The last item is the current page — rendered without a link.

## HTML structure

```html
<ol class="breadcrumb" aria-label="Breadcrumb">
  <li><a href="/">Home</a></li>
  <li><a href="/jobs">Jobs</a></li>
  <li aria-current="page">Senior Product Designer</li>
</ol>
```
