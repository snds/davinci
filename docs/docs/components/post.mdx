# Post

Post is the central content unit of the Davinci feed. It renders a structured social post with author identity, timestamp, body text, optional media attachments, and a reaction bar.

## Anatomy

```
┌───────────────────────────────────────┐
│ [Avatar]  Author Name          [•••]  │  ← .post__header
│           Role · Company              │
│           2h                          │
├───────────────────────────────────────┤
│                                       │
│  Post body text goes here.            │  ← .post__body
│                                       │
├───────────────────────────────────────┤
│  👍 ❤️ 🎉 42          12 comments     │  ← .post__reactions
├───────────────────────────────────────┤
│  Like · Comment · Repost · Send       │  ← .post__actions
└───────────────────────────────────────┘
```

## Action states

<DocsCallout type="note">
Post body uses `white-space: pre-wrap` to preserve line breaks. Sanitize all user-submitted content before inserting into `.post__body` to prevent XSS.
</DocsCallout>
