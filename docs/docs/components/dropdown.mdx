# Dropdown

Dropdown is a positioned menu that opens below (or beside) a trigger. It uses `.dropdown` as the panel and `.dropdown__item` for each action. Nested: Button for the trigger.

## HTML structure

```html
<div style="position: relative;">
  <button class="btn btn--secondary">
    Options
    <span class="material-symbols-rounded">expand_more</span>
  </button>

  <!-- Toggle visibility with JS -->
  <div class="dropdown">
    <div class="dropdown__label">Actions</div>
    <button class="dropdown__item">
      <span class="material-symbols-rounded">edit</span>
      Edit
    </button>
    <div class="dropdown__separator"></div>
    <button class="dropdown__item dropdown__item--danger">
      <span class="material-symbols-rounded">delete</span>
      Delete
    </button>
  </div>
</div>
```

<DocsCallout type="info">
  Wrap the trigger and `.dropdown` in a `position: relative` container. Toggle `display: none` / `display: block` with JS. Add `aria-haspopup="menu"` and `aria-expanded` on the trigger.
</DocsCallout>
